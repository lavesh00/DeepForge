# DeepForge

Autonomous AI-powered software development platform

## Quick Start (Windows)

**ONE COMMAND SETUP:**

**PowerShell:**
```powershell
.\setup.bat
```

**Command Prompt:**
```batch
setup.bat
```

That's it. The script will:
- Check Python installation
- Create virtual environment
- Install all dependencies (including transformers, torch, huggingface-hub)
- Create all required directories
- Verify installation

**START THE SERVER:**

**PowerShell:**
```powershell
.\setup.bat start
```

**Command Prompt:**
```batch
setup.bat start
```

Then open http://localhost:8080 in your browser.
